Slide-Menu

### slide-menu

Thank for the css style to http://demos.uilang.com/hamburger-menu/

Here show how to do the same with AniJS library, is really easy.

```html
<a href=# data-anijs="if: click, do: $toggleClass open">
        <span></span>
        <span></span>
        <span></span>
</a>
<nav></nav>
```
